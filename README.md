This project is a comparative IPO analysis of major two-wheeler pure EV companies in Asia,i.e Ola electric (India) & Gogoro (Taiwan)
Analysed Draft Red Herring Prospectuses (DRHPs) to understand business models, revenue drivers, cost structures, and key risks. 
Conducted peer valuation using P/E and EV/EBITDA multiples along with historical financial trend analysis.
Interpreted key factors influencing IPO pricing and post-listing performance.
